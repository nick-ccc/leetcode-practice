# Merge k Sorted Lists

### Prompt:
```
You are given an array of k linked-lists lists, each linked-list is sorted in ascending order.

Merge all the linked-lists into one sorted linked-list and return it.
```


### Thinking:

Store first node of each linked list in heap, pop the head and add in the next element from that 
linked list. Will uses some hints and research online since I am still unfamiliar with the general
